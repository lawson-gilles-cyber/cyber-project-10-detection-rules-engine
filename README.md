# cyber-project-10-detection-rules-engine
Custom detection rules engine for log analysis (SIEM simulation)


## Context

SIEM systems rely on detection rules to identify threats.

This project implements a simple rule-based detection engine.

---

## Objective

* Create detection rules
* Apply rules to logs
* Generate alerts

---

## ⚙️ Rules

* Brute Force → multiple failed logins
* Suspicious Login → success after failures
* Sensitive File Access → critical file event

---

## Key Takeaways

* Rule-based detection
* SIEM logic
* Cybersecurity automation
